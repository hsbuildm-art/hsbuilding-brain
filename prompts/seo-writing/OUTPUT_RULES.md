# SEO/AIO ライティング 出力ルール v1.0
> 最終更新: 2026-02-18
> ルール: 記事を制作する際、以下を必ずセットで一括出力する

## 必須出力セット（記事本文と同時に必ず生成）

### 1. メタタイトル（title タグ）
- 32文字以内推奨（最大60文字）
- メインKWを含む
- クリックされやすい表現（数字、年号、比較、最安 等）

### 2. メタディスクリプション（meta description）
- 120文字以内
- メインKW + サブKWを自然に含む
- 記事の結論を要約し、クリックを促す

### 3. URLスラッグ（slug）
- 英語のケバブケース（例: nara-virtual-office-cheapest-2026）
- 短く、KWを含む
- 日本語不可

### 4. 構造化マークアップ（JSON-LD）
- Article スキーマ（必須）
- 著者情報（キャラクター名 + HSビルワーキングスペース）
- 公開日・更新日
- 画像URL
- FAQPage スキーマ（記事内のFAQセクションと完全一致、最低3問・推奨5問）

### 5. 画像生成プロンプト
- アイキャッチ画像（1200x630）のプロンプト
- 本文挿入画像（800x450）のプロンプト（2〜4枚分）
- 担当キャラクターのIdentity Lockプロンプトを必ず含む

## 出力フォーマット

記事制作時は以下の順序で一括出力する:

1. メタ情報（タイトル、ディスクリプション、スラッグ）
2. 記事本文（Markdown）
3. 構造化マークアップ（Article + FAQPage の JSON-LD）
4. 画像生成プロンプト（アイキャッチ + 本文画像）
5. 配信テキスト（Instagram / note / LINE / X）

## 注意事項
- 記事本文だけの出力は禁止。必ず上記セットで出力する
- クーポンコード WELCOME10 をCTAに必ず含める
- 内部リンクは最低2本（予約ページ + 関連記事）
- キャラクタープロフィールを記事末尾に必ず含める
