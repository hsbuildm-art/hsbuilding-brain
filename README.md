# HSビル ワーキングスペース — ナレッジベース（Public）

## 概要
HSビルワーキングスペースの公開ナレッジベースです。
AI検索エンジン・外部パートナー・AIエージェントが参照する情報を管理しています。

## 公式サイト
https://www.hsworking.com

## LLM向け情報
https://www.hsworking.com/llm.txt

## このリポジトリの内容
- `knowledge/` — 施設・料金・アクセス・ビジョンなどの公開情報
- `ai-staff/` — AIスタッフ（マルモ・エリカ・ツバサ）のキャラクター情報
- `content/` — ブログ・SNSテンプレート
- `prompts/` — 汎用プロンプトガイドライン

## 非公開情報について
以下の情報は別リポジトリ（private）で管理しています：
- `.claude/rules/` — AI運用ルール
- `ai-staff/*/prompts.md` — AIスタッフ応答プロンプト
- `knowledge/linebot-architecture.md` — LINE Bot設計書
- `knowledge/external-services.md` — API設定・外部サービス連携
- `operations/` — 内部運用フロー・エスカレーション
- `consulting/` — コンサル提案書・ROIモデル
- `systems/` — システムアーキテクチャ
- `tsubasa/` — SEO/AIO診断ソースコード
- `wix/` — Wixバックエンドコード
- `drafts/` — 未公開記事

Private repo: `hsbuilding-brain-private`（同一GitHub org内）
