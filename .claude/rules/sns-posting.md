# SNS投稿ルール

## 適用範囲
```
content/sns/**/*.md
prompts/sns-posting/**/*.md
```

## 基本方針

### ブランドトーン
- 親しみやすく、でも専門性を感じさせる
- 前向きでエネルギッシュ
- 地域密着とグローバルな視点のバランス

### 投稿頻度
- X（Twitter）: 1日1〜2回
- Instagram: 週3〜4回
- Facebook: 週2〜3回
- LinkedIn: 週1〜2回

### コンテンツミックス
- 施設紹介: 30%
- お客様の声・事例: 20%
- AIスタッフコンテンツ: 20%
- ビジネスTips: 15%
- イベント・キャンペーン: 10%
- その他（地域情報等）: 5%

## プラットフォーム別ルール

### X（Twitter）
- **文字数**: 140〜280文字（簡潔に）
- **ハッシュタグ**: 2〜3個
- **画像**: 1〜4枚（推奨: 1〜2枚）
- **トーン**: カジュアル、リアルタイム性を重視

### Instagram
- **文字数**: キャプション100〜200文字程度
- **ハッシュタグ**: 10〜20個（コメント欄に記載）
- **画像**: 高品質、統一感のあるビジュアル
- **トーン**: ビジュアル重視、ライフスタイル提案

### Facebook
- **文字数**: 100〜300文字
- **ハッシュタグ**: 1〜3個
- **画像/動画**: 情報量の多いコンテンツも可
- **トーン**: 丁寧、コミュニティ重視

### LinkedIn
- **文字数**: 150〜300文字
- **ハッシュタグ**: 3〜5個（ビジネス関連）
- **画像**: プロフェッショナルなビジュアル
- **トーン**: ビジネス専門的、知見の共有

## AIスタッフ投稿
各AIスタッフのキャラクター性を活かした投稿：

### エリカ
- 施設案内、イベント告知
- 明るく歓迎的なトーン
- 参照: `ai-staff/erika/persona.md`

### ツバサ
- コワーキングTips、利用者目線の情報
- フレンドリーで実用的
- 参照: `ai-staff/tsubasa/persona.md`

### マルモ
- AI・ビジネストレンド、導入事例
- 専門的だが親しみやすく
- 参照: `ai-staff/marumo/persona.md`

## ハッシュタグ戦略

### 定番ハッシュタグ
```
#HSビル
#神戸コワーキング
#三宮オフィス
#神戸起業
#コワーキングスペース
#シェアオフィス
#バーチャルオフィス
```

### テーマ別ハッシュタグ
- AI関連: `#AI導入 #AIヘルプデスク #ChatGPT活用`
- 働き方: `#リモートワーク #フリーランス #ノマドワーク`
- 地域: `#神戸 #三宮 #兵庫県 #関西`

## 投稿時間帯（推奨）
- **平日朝**: 7:00〜9:00（通勤時間）
- **平日昼**: 12:00〜13:00（ランチタイム）
- **平日夕方**: 18:00〜20:00（退勤時間）
- **週末**: 10:00〜12:00

## 画像・動画ガイドライン
- **解像度**: 1080×1080px（Instagram）、1200×630px（その他）
- **ファイル形式**: JPG、PNG、MP4
- **ブランドカラー**: 統一感を保つ
- **テキストオーバーレイ**: 画像の20%以内

## エンゲージメント促進
- 質問形式の投稿
- 投票・アンケート機能の活用
- ユーザー生成コンテンツ（UGC）の再投稿
- コメントへの迅速な返信（24時間以内）

## 禁止事項
- 政治的・宗教的な主張
- 競合他社の誹謗中傷
- 炎上を招く可能性のある内容
- 不確かな情報の拡散
- 過度な宣伝・売り込み

## 投稿前チェックリスト
- [ ] ブランドトーンに合っているか
- [ ] 誤字脱字がないか
- [ ] ハッシュタグは適切か
- [ ] 画像の品質は十分か
- [ ] リンクは正しく機能するか
- [ ] キャラクター性（AIスタッフ投稿の場合）は一貫しているか
- [ ] 炎上リスクはないか

## 効果測定
- エンゲージメント率
- リーチ数
- フォロワー増加数
- Webサイトへの流入数
- 問い合わせ・予約への貢献
