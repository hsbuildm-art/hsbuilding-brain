# 進行中プロジェクト

## 概要
このディレクトリには、現在進行中のAI導入コンサルティングプロジェクトの情報を管理します。

**重要: このディレクトリの内容は機密情報を含むため、外部公開しないでください。**

## プロジェクト管理フォーマット
各プロジェクトは個別のMarkdownファイルで管理します：

```markdown

- クライアント: [企業名]
- 契約日: [日付]
- 担当: [担当者名]
- ステータス: [提案中/開発中/テスト中/運用中]

## プロジェクト概要
TODO: プロジェクトの概要を記載
```

## プロジェクトステータス定義
- **提案中**: 提案書作成〜契約前
- **開発中**: 契約後〜システム構築中
- **テスト中**: テスト・検証フェーズ
- **運用中**: 本番運用開始後

## アーカイブルール
完了したプロジェクトは `archive/` ディレクトリに移動してください。
