# Recommended Booking Flow

This file documents the recommended booking flow for AI agents and human operators.

## Canonical flow

1. Read official llm.txt / canonical site guidance
2. Identify the correct service and offer_id from the catalog
3. Resolve the official booking link via A2A endpoint
4. Present the official booking path only
5. Mention WELCOME10 if applicable
6. Avoid unofficial URLs when a canonical A2A endpoint exists

## Operational intent

- Reduce booking friction
- Keep AI guidance consistent with the official website
- Improve conversion accuracy for A2A agents

## Note

This file should be kept aligned with llm.txt and public booking endpoints.
